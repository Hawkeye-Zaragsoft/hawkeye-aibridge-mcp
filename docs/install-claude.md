# Install Hawkeye AI Bridge in Claude Desktop

This is a placeholder guide for connecting Hawkeye AI Bridge to Claude Desktop through MCP.

## Example configuration

```json
{
  "mcpServers": {
    "hawkeye": {
      "command": "C:\\Program Files\\Hawkeye\\HawkeyeAIBridge.exe",
      "args": []
    }
  }
}
```

Adjust the executable path based on where Hawkeye AI Bridge is installed.
