# Security Policy

If you discover a security issue related to Hawkeye AI Bridge or its MCP integration, please contact:

info@zaragsoft.se

Please do not disclose security issues publicly before Zaragsoft has had a reasonable opportunity to investigate and respond.

## Notes

Hawkeye AI Bridge is intended to run locally and connect MCP-compatible clients to Hawkeye functionality on the user's machine.

Users should only configure MCP clients they trust and should review AI-generated changes before applying them.
